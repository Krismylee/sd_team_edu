# 02. 검색-후처리-결합 흐름

## 이 챕터에서 배우는 것

- 다중 검색 결과를 처리하는 표준 흐름
- 후처리와 결합의 경계
- 실무 적용을 위한 체크포인트

---

## 1. 표준 흐름

1) 다중 검색 수행
2) 검색 결과 정규화
3) 1차 후처리(필터링/재정렬)
4) 결과 병합
5) 2차 후처리(압축/정제)

---

## 2. 정규화의 중요성

검색 전략이 다르면 점수 스케일이 다릅니다.
정규화 없이 병합하면 특정 전략에 결과가 몰릴 수 있습니다.

---

## 3. 후처리와 결합의 경계

- 후처리 1: **전략별 품질 보정**
- 결합: **다양한 전략 결과 통합**
- 후처리 2: **생성 최적화**

---

## 4. 적용 시 주의점

- 중복 제거 기준이 모호하면 **품질 흔들림** 발생
- 병합 후 Top-K 제한을 반드시 적용
- 컨텍스트 길이 제한을 고려한 압축 필요

---

## 5. 체크리스트

- 정규화 규칙이 명확한가?
- 후처리 단계가 중복되지 않는가?
- 최종 컨텍스트 길이를 관리하고 있는가?
