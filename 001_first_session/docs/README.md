# 문서 안내서

이 문서는 LangChain/LangGraph 입문자를 대상으로 한 교육 자료 모음입니다. 각 문서는 "한 주제-여러 챕터" 구조로 구성되어 있으며, 실무에서 바로 적용할 수 있도록 **설명 + 이유 + 예제 + 코드**가 함께 제공됩니다.

## 디렉터리 구조

- `docs/01_prompt_engineering`: 프롬프트 엔지니어링 기초와 응용
- `docs/02_langgraph_basics`: LangGraph 기본 원칙과 설계 패턴
- `docs/03_llm_text_handling`: LLM 텍스트 처리 파이프라인과 전략
- `docs/04_string_tricks`: YES/NO, 단일/복수 선택, JSON 파싱 등 문자열 테크닉

## 사용 방법

1. 먼저 `01_prompt_engineering`부터 읽어 프롬프트 설계의 원칙을 이해합니다.
2. `02_langgraph_basics`에서 태스크 분리와 그래프 설계를 학습합니다.
3. `03_llm_text_handling`에서 문자열 처리의 전체 전략을 익힙니다.
4. `04_string_tricks`에서 실제 케이스별 파싱/정규화 코드를 확인합니다.

각 문서는 초보 개발자를 대상으로 하므로 **왜 그런 설계를 해야 하는지**와 **어떻게 구현해야 하는지**를 최대한 자세히 설명합니다.

## 구현 체크리스트

- 문서를 읽는 순서를 정하고 팀 내 합의된 기준을 만든다.
- 프롬프트 출력 형식을 먼저 고정하고 파싱/검증 규칙을 함께 작성한다.
- 실패 시 재요청/보수적 기본값/휴먼 리뷰 경로를 명확히 정한다.
- 샘플 코드를 프로젝트 규칙에 맞게 경량화하고 재사용 가능한 구조로 만든다.
